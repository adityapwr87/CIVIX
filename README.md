 # CIVIX - Civic Issue Management Platform

CIVIX is a full-stack web platform that helps citizens report, track, and resolve civic issues with real-time updates.  
It makes it easier for people to raise concerns and for local authorities to respond quickly.

##  What it does
- Citizens can **report civic issues** (like potholes, garbage, etc.) with images and location.  
- Admins can **track and manage issues**, update statuses, and resolve them.  
- **Real-time chat** between users and admins makes communication easy.  
- Issues and profiles support **image uploads** using cloud storage.  

---

##  Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (role-based: users, admins)  
- **Realtime:** Socket.IO  
- **Storage:** AWS S3  

---

##  Features
- Secure login/signup with JWT  
- Report issues with title, description, image, and location  
- Role-based dashboards (citizens & district admins)  
- REST APIs for managing issues and users  
- Real-time updates and chat
